# Panduan Pengguna Atmando Health

## Cara Akses Aplikasi

1. Buka browser di HP atau komputer
2. Kunjungi: `https://atmando-health.vercel.app`
3. Login dengan akun Google atau magic link via email
4. Pilih anggota keluarga yang ingin dilihat/dikelola

## Fitur Utama

### 1. Data Kesehatan (Health Metrics)

Catat dan pantau data kesehatan seperti:
- Berat badan
- Tinggi badan
- Tekanan darah
- Gula darah
- Detak jantung
- Suhu tubuh

**Cara menambah data:**
1. Pilih anggota keluarga
2. Tap tombol "+" atau "Tambah Data"
3. Pilih jenis pengukuran
4. Masukkan nilai
5. Tap "Simpan"

### 2. Dokumen Medis

Simpan dokumen penting seperti:
- Hasil lab
- Resep dokter
- Rekam medis
- Foto rontgen/USG
- Kartu vaksinasi
- Kartu asuransi

**Cara upload dokumen:**
1. Pilih anggota keluarga
2. Buka menu "Dokumen"
3. Tap "Upload"
4. Pilih file (PDF atau foto)
5. Isi keterangan (tanggal, dokter, rumah sakit)
6. Tap "Simpan"

### 3. Obat-obatan

Catat obat yang sedang dikonsumsi:
- Nama obat dan dosis
- Jadwal minum
- Instruksi (sebelum/sesudah makan)
- Tanggal mulai dan selesai

**Cara menandai obat sudah diminum:**
1. Buka menu "Obat"
2. Lihat daftar obat hari ini
3. Tap centang pada obat yang sudah diminum

### 4. Kunjungan Dokter

Catat riwayat kunjungan ke dokter:
- Tanggal kunjungan
- Nama dokter dan spesialisasi
- Rumah sakit/klinik
- Keluhan dan diagnosis
- Pengobatan yang diberikan
- Jadwal kontrol berikutnya

## FAQ (Pertanyaan Umum)

**Q: Siapa yang bisa melihat data kesehatan?**
A: Hanya anggota keluarga yang terdaftar. Dio dan Celline bisa melihat semua data. Anak-anak hanya bisa melihat data sendiri.

**Q: Apakah data aman?**
A: Ya, semua data dienkripsi dan disimpan di server yang aman. Hanya keluarga yang bisa akses.

**Q: Bagaimana jika lupa password?**
A: Gunakan opsi "Magic Link" - link login akan dikirim ke email.

**Q: Bisa diakses offline?**
A: Untuk saat ini perlu koneksi internet. Fitur offline akan ditambahkan nanti.

**Q: Format file apa yang didukung?**
A: PDF, JPG, PNG. Maksimal 10MB per file.

## Troubleshooting

**Tidak bisa login:**
- Pastikan email yang digunakan sudah terdaftar
- Coba gunakan magic link jika Google login bermasalah
- Hubungi Dio jika masih bermasalah

**Upload dokumen gagal:**
- Pastikan ukuran file tidak lebih dari 10MB
- Pastikan format file PDF, JPG, atau PNG
- Coba kompres file jika terlalu besar

**Data tidak muncul:**
- Refresh halaman (tarik ke bawah di HP)
- Pastikan koneksi internet stabil
- Coba logout dan login kembali

## Kontak

Jika ada masalah atau pertanyaan, hubungi Dio.
